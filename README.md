# Spotify Data Analysis

## 📌 Objective

Analyze Spotify dataset to find trends in music popularity.

## 📊 Dataset

spotify_2023.csv

## 🛠️ Tools Used

* Python
* Pandas
* Seaborn
* Matplotlib

## 🔍 Steps Performed

* Loaded dataset using pandas
* Cleaned missing values
* Analyzed popularity trends
* Visualized data using graphs

## 📈 Insights

* Most songs have medium popularity
* Some artists dominate the charts
* Energy affects popularity

## 🚀 Conclusion

This project helps understand music trends using data analysis.
